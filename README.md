# 🚀 Baseera Pro - Production Ready

منصة Baseera Pro هي حل متكامل للنشر الذكي عبر منصة Baseera Intelligent Automator.

## الحالة
الحالة: نشط 🟢

## نظرة عامة

Baseera Pro هو مشروع ويب متكامل يوفر واجهة مستخدم احترافية مع ميزات متقدمة للنشر والتشغيل.

## التقنيات المستخدمة

- **Frontend**: HTML, JavaScript, Tailwind CSS
- **Build Tool**: Vite
- **Package Manager**: pnpm
- **Deployment**: Vercel

## الملفات الرئيسية

- [`index.html`](./index.html) - الصفحة الرئيسية للتطبيق
- [`src/`](./src/) - الكود المصدري للمشروع
- [`services/`](./services/) - الخدمات والمسارات البرمجية
- [`vite.config.ts`](./vite.config.ts) - إعدادات Vite
- [`tailwind.config.js`](./tailwind.config.js) - إعدادات Tailwind CSS
- [`package.json`](./package.json) - تبعيات المشروع

## التثبيت

```bash
# استنساخ المشروع
git clone <repository-url>

# تثبيت التبعيات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# البناء للإنتاج
pnpm build
```
